계정 및 데이터 삭제

잇데이 스튜디오 (Itdasy Studio) · 와이투두(Y2do) · 최종 갱신: 2026-04-23

잇데이 스튜디오 앱의 계정 삭제를 요청하는 방법을 안내드립니다.

1. 앱 내에서 계정 삭제 (권장)

  1. 잇데이 스튜디오 앱 실행 → 로그인
  2. 우측 상단 ⚙ 설정 아이콘 탭
  3. 하단 "회원 탈퇴" 메뉴 선택
  4. 안내 확인 후 "탈퇴 진행" 탭
  5. 2차 확인 모달에서 "정말 탈퇴" 선택

탈퇴 즉시 계정 · 고객 · 예약 · 매출 · 사진 등 모든 개인 데이터가 삭제됩니다.

2. 이메일로 삭제 요청

앱 접속이 어려우시면 contact@itdasy.com 으로 아래 정보와 함께 요청해주세요:

  • 가입 이메일 주소
  • 샵 이름 (확인용)
  • 요청 사유 (선택)

영업일 기준 3일 이내 처리합니다.

3. 삭제되는 데이터

계정 삭제 시 다음 데이터가 영구 삭제됩니다:

개인정보 이메일 · 이름 · 비밀번호 해시 · 프로필 사진

앱 데이터 고객 목록 · 예약 · 매출 기록 · 시술 사진 · 캡션 · 페르소나 설정

연동 정보 Instagram 토큰 (저장된 것 모두 암호화 삭제)

로그 본인 user_id 관련 Sentry 오류 로그 (90일 이내 자동 정리)

4. 보관되는 데이터 (법령 준수)

전자상거래법 · 세법 등 국내 법령 준수를 위해 아래 데이터는 일정 기간 보관 후 자동 삭제됩니다:

  • 결제 기록 (IAP): 5년 (전자상거래법)
  • 로그인 IP 및 접속 기록: 3개월 (통신비밀보호법)
  • 부정이용 방지 로그: 1년

보관 기간 만료 후에는 복원 불가능하게 폐기됩니다. 해당 데이터는 암호화 저장되며 개인 식별용 외엔 접근 불가합니다.

5. 일부 데이터만 삭제

계정 유지한 채 특정 데이터만 삭제하고 싶으시면:

  • 고객 개별 삭제: 고객 목록 → 해당 고객 → 휴지통
  • 매출 기록 삭제: 매출 탭 → 해당 기록 스와이프 → 삭제
  • 예약 삭제: 예약 캘린더 → 예약 탭 → 삭제
  • 사진 삭제: 작업실 → 슬롯 → 사진 삭제
  • Instagram 연동 해제: 설정 → 연동 관리 → 해제

6. 외부 위탁사 데이터 삭제

계정 삭제 시 외부 서비스 제공사에도 자동으로 삭제 요청이 전송됩니다:

  • Google Gemini API: stateless 호출 — 저장 데이터 없음
  • Meta (Instagram): /instagram/data-deletion 콜백으로 연동 해제
  • Replicate (이미지 처리): 1시간 후 자동 폐기
  • Supabase (DB): 즉시 CASCADE 삭제
  • Sentry (로그): user_id 스크럽 요청
