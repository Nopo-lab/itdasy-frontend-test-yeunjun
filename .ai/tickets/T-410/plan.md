# T-410 Plan · Sprint E CRM/재고 통합

## 신호등

🟡 노란불 — 스테이징 프론트 기능 보강. 운영 레포/운영 DB/결제/개인정보 직접 변경 없음.

## 할 일

1. `app-complete-flow.js`
   - 예약 완료 팝업에서 예약 금액 또는 시술 프리셋 금액을 자동 입력한다.
   - 완료/매출 저장 후 `itdasy:data-changed` 신호를 보내 고객·매출·예약 화면이 갱신되게 한다.
   - 예약 상태 변경은 가능하면 `window.Booking.update()`를 써서 예약 캐시도 같이 비운다.

2. `app-calendar-view.js`
   - 예약 생성/수정/삭제 후에도 공통 변경 신호를 보낸다.

3. `app-inventory-hub.js`
   - 재고 추가/묶음 저장/수량 변경/수정 후 공통 변경 신호를 보낸다.

4. `app-myshop-v3.js`
   - 내샵관리 화면이 변경 신호를 받으면 오래된 요약을 버리고 다시 불러오게 한다.

5. 검증
   - 문법 확인
   - 자동 확인
   - 이번 변경 파일 중심 검사
