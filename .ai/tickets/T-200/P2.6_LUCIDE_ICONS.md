# T-200 P2.6 · Lucide 아이콘 통일 (네비·헤더·테마 토글 범위)

> **상태:** 🔴 원영님 재스코프 승인 대기
> **선결 조건:** P2.5 (하단 네비) 완료
> **담당:** T2 FE Coder

## 왜 필요한가

원영님 지적: "위 달덩이 뭐냐. 루시드로 같이 통일해." 현 테마 토글은 `🌗` 이모지. 프로토 v2 는 전 화면 Lucide SVG.

## 범위 (IN)

1. **테마 토글:** `🌗` → 프로토 v2 L206 SVG moon (`M21 12.79A9 9 0 1 1 11.21 3 7 7 0 0 0 21 12.79z`)
2. **하단 네비 5버튼** (P2.5 에서 `<span>` 만 넣었으면 여기서 SVG 추가):
   - 홈: house path (프로토 L344)
   - 만들기(=작업실): rect+십자 path (프로토 L348)
   - AI추천: 별 path (프로토 L352)
   - 마무리: 체크 path (프로토 L356)
   - 내 샵(=페르소나): 가방/쇼핑백 path (프로토 L360)
3. **상단 헤더 알림 벨:** 프로토 L220 bell path (현재 없음, 신설)

## 범위 (OUT)

- 본문 내 이모지 (예: `💬 고객지원`, `✨ 만들기`) 는 이번 범위 밖. 별도 티켓 T-202 (본문 아이콘 정비) 로 분리.
- 작업실 내부 이모지 (프로토에 없는 영역) 은 해당 탭 재작업 Phase 에서 처리.

## 아이콘 규칙 (본 문서로 확정)

- **UI(네비·헤더·버튼 아이콘·테마 토글)** = Lucide SVG **only**. 이모지 금지.
- **본문 텍스트** (인사말, 토스트 메시지, 채팅 말풍선) = 이모지 허용.
- Lucide 스펙: `viewBox="0 0 24 24"`, `stroke="currentColor"`, `stroke-width="1.8"~"2"`, `stroke-linecap="round"`, `stroke-linejoin="round"`, `fill="none"` 기본.
