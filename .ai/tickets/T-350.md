# T-350 · 앱 심사 통과 + 법적 리스크 일괄 강화

**상태:** 구현 완료 (2026-04-22)
**트랙:** 표준 (회귀 리스크 있음 — 약관·방침·동의 플로우)

## 목적
Apple/Google 심사 반려 위험 및 국내 법적 노출을 한 번에 해소.

## 구현 내용

### A. 계정 탈퇴 인앱 노출 (Apple 5.1.1(ix))
- `index.html`: 설정 시트에 🗑 **계정 탈퇴** 버튼 명시 노출 + 설명 텍스트
- `#deleteAccountModal`: 2단계 확인 모달 — "탈퇴" 텍스트 입력 + `nativeConfirm` 최종 확인
- `app-core.js`: `openDeleteAccountModal()`, `closeDeleteAccountModal()`, `confirmDeleteAccount()` 핸들러. 기존 `DELETE /auth/delete-account` BE 엔드포인트 호출
- 성공 시 `localStorage.clear()` + Cache API 전부 삭제 + index.html 이동

### B. AI 생성 콘텐츠 신고 (Apple 1.5 / UGC)
- BE 신규: `routers/moderation.py` — `POST /moderation/report`, `GET /moderation/categories`
  - 8개 카테고리 (offensive/sexual/violence/misinformation/privacy/copyright/spam/other)
  - 티켓 ID `MOD-YYYYMMDDHHMMSS-U{user_id}` 발급
  - Discord 웹훅 알림 (`DISCORD_MODERATION_WEBHOOK` or 기본 웹훅)
- `main.py`에 router include
- FE 신규: `app-content-report.js` — `window.openContentReport()` + `data-report-ai` 자동 바인딩
- 캡션 액션바에 🚩 신고 버튼 (`app-caption.js`)
- 챗봇 답변 버블에 🚩 신고 버튼 (`app-assistant.js`)

### C. 개인정보처리방침 대폭 강화 (`itdasy-promo/privacy.html`)
- 4조 정보주체 권리: PIPA 제4조·제35~37조 명문화, 인앱 탈퇴권·이동권 명시
- 6조 쿠키·자동수집: 전체 저장소 테이블(5개) + 삭제 방법
- **6조의2 신규 — 자동화된 의사결정(AI 처리 고지)**: Gemini API 사용 범위, 가명처리, 거부권
- 7조 보호책임자 + 4개 분쟁 조정 기관 연락처 (분쟁조정위·KISA·검찰·경찰)
- **9조 신규 — 유출 통지 절차**: 72시간 통지, 1000명+ KISA 신고, 손해배상
- 10조 개정 이력 테이블 + 버전 관리
- **`privacy-en.html` 신규** — 영문 정책 (App Store Connect 제출용)

### D. 이용약관 AI 면책 강화 (`itdasy-promo/terms.html`)
- 7조 결제: IAP 명문화, 플랜별 가격, 자동갱신·해지 절차
- 10조 면책 항목 6개로 확장 (Apple·Google 정책 변경 포함), 유료 책임 한도 3개월
- **10조의2 신규 — AI 생성물 면책**: 7개 조항 (보조 역할, 정확성 미보증, 검토 의무 4항목, 법적 책임 귀속, 신고 24시간, 금지 용도, AI 거부권)
- 14조 일부 무효 처리, 15조 통지 방법 신규, 16조 기타
- 개정 이력 테이블 + `terms-en.html` 신규

### E. Apple Privacy Labels 질문지 (`docs/submission/Apple-Privacy-Labels.md`)
- App Store Connect Privacy Questionnaire 모든 섹션 답변
- 데이터 타입별 Linked/Tracking 분류
- Third-Party SDKs (Gemini/Meta/Replicate/Cloudflare/Supabase/Railway/Sentry) 목록
- IDFA 미사용 선언 · ITSAppUsesNonExemptEncryption=false 근거

### F. Gemini Safety Settings 보수적 설정 (`services/generation.py`)
- `HARM_CATEGORY_HARASSMENT/HATE_SPEECH/SEXUALLY_EXPLICIT/DANGEROUS_CONTENT` → `BLOCK_MEDIUM_AND_ABOVE`
- 모든 generate_json/text 함수 config 에 적용
- Apple 1.1 / 1.2 harmful content 보호
