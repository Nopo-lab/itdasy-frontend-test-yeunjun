**근거:**
- `app-core.js:268-279` 의 `getToken()` 이 canonical getter. `_TOKEN_KEY` 사용 + JWT 만료 체크까지 포함
- `getToken()` 은 top-level function 이라 classic script 로드 시 `window.getToken` 에 자동 노출
- `window.getToken` 존재 여부 guard 는 core.js 로드 전에 push 가 먼저 실행되는 엣지 케이스 대비
