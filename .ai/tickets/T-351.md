# T-351 · 해외 출시 대응 — GDPR/CCPA/LGPD/APPI/PIPEDA/호주법 일괄 준수

**상태:** 구현 완료 (2026-04-22)
**트랙:** 표준 — 법적 문서 + FE 동의 배너 + BE 포터빌리티

## 목적
Apple/Google Store 글로벌 출시 후 EU·미국·브라질·일본·캐나다·호주 등 해외 사용자 수용 시 즉시 법적 컴플라이언스 달성.

## 구현 내용

### A. Privacy Policy 국제법 대응
**`itdasy-promo/privacy.html`** (v1.1)
- 11조 신설 — 해외 거주자 고지 (GDPR/CCPA/기타)
- EU/UK: 법적 근거·권리·SCCs·감독기관
- 캘리포니아: No Sell/Share 선언, CCPA 권리, Shine the Light
- 기타국: LGPD(브라질), APPI(일본), PIPEDA(캐나다), Privacy Act(호주)
- 아동 보호 multi-jurisdictional (PIPA 14·COPPA 13·GDPR 16·CPRA 16)
- 쿠키/ePrivacy 고지

**`itdasy-promo/privacy-en.html`** (v1.1) — 풀 영문판
- Sections 11~19 신설, 각 법령별 상세 권리/절차/감독기관

### B. Terms 국제 소비자법
**`itdasy-promo/terms.html`** (v1.1)
- 16조 신설 — 해외 이용자 고지
- EU 14일 철회권 (Directive 2011/83/EU)
- 미국 JAMS 구속력 중재 + 집단소송 배제 + 30일 opt-out
- 브라질·호주·캐나다·일본 강행 소비자법 존중
- 수출 통제·제재 준수, 접근성(WCAG 2.1 AA)

**`itdasy-promo/terms-en.html`** (v1.1) — 풀 영문판 Articles 14~19

### C. 쿠키/분석 동의 배너 (GDPR ePrivacy)
**`itdasy-frontend-test-yeunjun/app-cookie-consent.js` 신규**
- Timezone 기반 EU 탐지 (IP 판별 없이, 프라이버시 존중)
- EU 사용자: opt-in 필수 (Sentry 기본 비활성)
- 기타 지역: 기본 허용 + 배너 노출
- `window.itdasyConsent.isAnalyticsAllowed()` 공개 API
- `index.html`: Sentry 초기화 전 consent 확인 가드 추가

### D. 데이터 포터빌리티 (GDPR Art. 20 / CCPA / PIPA §35)
**`itdasy_backend/backend/routers/data_export.py` 신규**
- `GET /data-export/summary` — 건수 프리뷰
- `GET /data-export/json` — 기계 판독 JSON 다운로드
- `GET /data-export/csv` — 테이블별 CSV zip 다운로드
- 비밀번호·토큰·암호 자동 제외
- `main.py`에 router include

**`app-data-export.js` 신규**
- 설정 시트에 "📦 내 데이터 내보내기" 버튼
- CSV(zip) / JSON 두 포맷 제공
- 건수 프리뷰 → 다운로드 트리거

### E. 컴플라이언스 문서
`itdasy-frontend-test-yeunjun/docs/legal/` 신규
- **DPA-Template.md** — GDPR Art. 28 DPA 템플릿 (15개 조항, 엔터프라이즈 계약용)
- **DPIA-Summary.md** — GDPR Art. 35 영향평가 (리스크 매트릭스 + 완화책)
- **RoPA.md** — Art. 30 처리 활동 기록 (Controller/Processor 이중 역할)
