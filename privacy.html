2. 개인정보 처리 목적

  • 회원 식별 및 인증, 비밀번호 재설정
  • 맞춤형 캡션 생성 (페르소나 톤·해시태그·이모지 추출)
  • 이미지 누끼/배경 합성 등 AI 처리 결과 제공
  • 인스타그램 게시물 분석 및 발행 보조
  • 인스타그램 DM 자동응답: 받은 DM 카테고리 분류 → AI 답장 초안 생성 → 사장님 검토 큐(confirm queue) → 승인 후 발송. 사장님 톤 학습 (sample_replies)
  • 인스타그램 댓글 관리: 받은 댓글 모니터링, 자동 답글 (해당 권한 승인 시 활성)
  • 매장 운영 관리: 예약(월/주/일 캘린더 + drag-drop), 매출(결제방식별 도넛 + 순수익 계산), 고객(이탈 위험 분석 + 단골 관리), 재고(부족 알림), 회원권(잔액·만료 안내)
  • 카카오 알림톡 발송(준비 중): 예약확정/리마인드/생일 자동 메시지
  • 구독 관리, 결제 이력 보관 (전자상거래법 의무)
  • 부정 이용 방지, 보안 사고 대응 (IP 로그)

5. 개인정보 파기 절차 및 방법

  • 파기 시점: 회원이 "회원 탈퇴"를 요청하면 즉시 모든 개인정보 및 연관 데이터를 파기합니다 (보유 의무가 있는 결제 이력 제외).
  • 파기 방법:
    • DB: 15개 user-linked 테이블 행 즉시 삭제 (CASCADE) + User 행 익명화 (이메일/이름/비밀번호 해시 모두 비움)
    • 파일: 사용자 업로드 디렉토리(`users/{user_id}/`) 즉시 영구 삭제
    • 인스타그램 토큰: Fernet 암호화 칼럼 삭제
    • 외부 위탁사 보유분: Meta/Replicate/Remove.bg 등은 자체 정책에 따라 처리 (요청 시 본 서비스가 대신 삭제 요청 진행)
  • 감사 로그: 파기 사실은 `UserDeletion` 테이블에 1년간 기록 (분쟁 대응 목적, 식별정보 미포함).

6. 정보주체의 권리 행사 방법

  • 정보주체는 언제든지 다음 권리를 행사할 수 있습니다: 열람, 정정·삭제, 처리정지, 동의 철회, 데이터 이동(포터빌리티).
  • 데이터 이동권 / 내보내기 (PIPA §35, GDPR Art.20, CCPA §1798.100(d)): 앱 내 "설정 > 백업 · 내보내기" 또는 "톱니바퀴 > 내 데이터 내보내기"에서 본인 데이터를 CSV(ZIP) / JSON 으로 즉시 다운로드. 포함 항목: 고객·예약·매출·재고·지출. 제외 항목: 비밀번호 해시·OAuth 토큰(IG/Kakao/Naver/Google/Apple)·결제 영수증 원본·외부 위탁사 보관 데이터.
  • 회원 탈퇴: 앱 내 "설정 > 회원 탈퇴" 또는 DELETE /auth/delete-account API. 탈퇴 시 모든 개인정보 즉시 익명화·삭제.
  • 인스타 연동 해제: 앱 내 "인스타 해제" 버튼 또는 Instagram 계정에서 본 서비스 권한 해제 → 24시간 내 Access Token + DM/댓글 + 페르소나 학습 데이터 자동 파기 (`/data-deletion` 페이지 참조).
  • AI 학습 사용 범위: 본 회원의 데이터는 본 회원의 캡션·DM 생성 품질 향상 목적으로만 사용되며, 제3자 광고 타겟팅·외부 AI 모델 학습용으로 사용되지 않습니다. 학습 데이터는 본 회원의 잇데이 계정 안에 격리됩니다.
  • 문의: 아래 책임자 연락처로 요청.

7. 개인정보의 안전성 확보 조치

  • 비밀번호: bcrypt 단방향 해시 저장 (원문 보관 안 함, 8자 이상 + 영문 + 숫자 정책)
  • 인스타그램 Access Token: Fernet 대칭키 암호화 저장 (`FERNET_KEY` 환경변수, 서버 부팅 시 누락 검증)
  • 전송: HTTPS (TLS 1.2 이상) 강제
  • 접근 통제: 관리자 전용 엔드포인트는 `is_admin` 권한 검증, JWT 토큰 만료 정책
  • 로그인 보호: 5분 내 10회 실패 시 차단(Rate Limit), IP+이메일 기록
  • 전역 보호: API 호출 IP당 분당 120회 제한
  • 모니터링: Sentry SDK로 비정상 접근 실시간 감지

8. AI 생성물의 저작권 및 책임

  • 본 서비스의 AI(Gemini 등)가 생성한 캡션·이미지의 상업적·비상업적 활용 권한은 사용자에게 귀속됩니다.
  • 현행법상 AI 단독 생성물은 저작권 보호 범위가 제한될 수 있습니다.
  • AI가 생성한 해시태그(예: #내돈내산, #협찬)나 시술 정보가 사실과 다를 경우 「표시·광고의 공정화에 관한 법률」 위반이 될 수 있으며, 최종 게시 전 사용자 본인이 검수할 의무가 있습니다.

9. 쿠키 및 로컬 스토리지
